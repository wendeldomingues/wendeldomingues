## 🚀 Sobre Mim  

Sou desenvolvedor **Back-End** com foco em construção de APIs, automação de processos e integração de sistemas utilizando Python.

Tenho experiência prática desenvolvendo soluções que conectam sistemas, tratam dados e automatizam fluxos complexos de negócio, sempre com foco em performance, organização e escalabilidade.

Atualmente, estou focado em evoluir como **Backend Engineer**, trabalhando com:

- 🔹 Construção de APIs REST robustas com FastAPI  
- 🔹 Modelagem e otimização de bancos de dados (PostgreSQL)  
- 🔹 Integração entre sistemas e processamento de dados  
- 🔹 Estruturação de projetos com boas práticas de arquitetura  
- 🔹 Deploy e infraestrutura com Docker e AWS  

Minha base em automação me permite enxergar problemas de forma prática e construir soluções eficientes, indo além do código e focando no impacto real no negócio.

Gosto de criar sistemas bem estruturados, fáceis de manter e preparados para crescer.
